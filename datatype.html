Data Types

What are Data Types?

Data types define the type and size of data that can be stored in a variable. Different programming languages support various data types such as numbers, strings, booleans, and more.

Common Data Types

  • Number
  • String
  • Boolean
  • Array
  • Object

Examples of Data Types

Here are some examples of different data types:

var number = 10;
      var text = "Hello, World!";
      var isTrue = true;
      var fruits = ["apple", "banana", "orange"];
      var person = { name: "John", age: 25 };

In the above examples:

  • number is a variable of type number that stores the value 10.
  • text is a variable of type string that stores the text "Hello, World!".
  • isTrue is a variable of type boolean that stores the value true.
  • fruits is a variable of type array that stores multiple values of type string.
  • person is a variable of type object that stores a collection of properties (name and age).

Example: Using Data Types in JavaScript

        // Example of using data types
var number = 42;
var name = "John Doe";
var isTrue = true;
var fruits = ["apple", "banana", "orange"];
var person = { firstName: "John", lastName: "Doe" };

console.log("Number: " + number);
console.log("Name: " + name);
console.log("Is True: " + isTrue);
console.log("Fruits: " + fruits);
console.log("Person: " + person.firstName + " " + person.lastName);
