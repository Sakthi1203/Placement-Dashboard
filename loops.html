Loops

For Loop

The following example demonstrates a for loop in JavaScript:

for (let i = 1; i <= 5; i++) {
    console.log(i);
  }

While Loop

The following example demonstrates a while loop in JavaScript:

let i = 1;
  while (i <= 5) {
    console.log(i);
    i++;
  }

Do-While Loop

The following example demonstrates a do-while loop in JavaScript:

let i = 1;
  do {
    console.log(i);
    i++;
  } while (i <= 5);
