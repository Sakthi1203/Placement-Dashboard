Operators in C

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations in C.

Example:

#include <stdio.h>

int main() {
  int a = 10;
  int b = 5;
  int sum = a + b;
  int product = a * b;
  
  printf("Sum: %d\n", sum);
  printf("Product: %d\n", product);
  
  return 0;
}

In the above example:

  • a and b are initialized with the values 10 and 5, respectively.
  • sum is assigned the sum of a and b (10 + 5).
  • product is assigned the product of a and b (10 * 5).
  • The printf function is used to print the values of sum and product.

Ternary Operator

The ternary operator (? :) is a shorthand way of writing an if-else statement.

Example:

#include <stdio.h>
  
  int main() {
    int age = 20;
    char* message = (age >= 18) ? "You are an adult." : "You are not an adult.";
    
    printf("%s\n", message);
    
    return 0;
  }

In the above example:

  • The variable age is initialized with the value 20.
  • The ternary operator is used to assign the value of message based on the condition age >= 18.
  • If the condition is true, the value "You are an adult." is assigned to message. Otherwise, the value "You are not an adult." is assigned.
  • The printf function is used to print the value of message.

Relational Operators

Relational operators are used to compare values in C.

Example:

#include <stdio.h>
  
  int main() {
    int a = 10;
    int b = 5;
    
    if (a > b) {
      printf("a is greater than b\n");
    } else {
      printf("a is not greater than b\n");
    }
    
    return 0;
  }

Logical Operators

Logical operators are used to perform logical operations in C.

Types

  • Logical OR(||)
  • Logical AND (&&)
  • Negation(!)

Example:

#include <stdio.h>
  
  int main() {
    int a = 5;
    int b = 10;
    
    if (a > 0 && b < 20) {
      printf("Both conditions are true\n");
    }
    
    if (a > 0 || b < 20) {
      printf("At least one condition is true\n");
    }
    
    if (!(a > 10)) {
      printf("Negation of condition is true\n");
    }
    
    return 0;
  }

Shift Operators

Shift operators are used to shift the bits of a value in C.

Example:

#include <stdio.h>
  
  int main() {
    int a = 5;  // Binary: 00000101
    
    int leftShift = a << 2;  // Binary: 00010100 (20 in decimal)
    int rightShift = a >> 2;  // Binary: 00000001 (1 in decimal)
    
    printf("Left Shift: %d\n", leftShift);
    printf("Right Shift: %d\n", rightShift);
    
    return 0;
  }

Assignment Operators

Assignment operators are used to assign values to variables.

Example:

#include <stdio.h>
  
  int main() {
    int x = 5;
    int y = 10;
    
    x += y; // Equivalent to: x = x + y;
    printf("x: %d\n", x);
    
    y *= 2; // Equivalent to: y = y * 2;
    printf("y: %d\n", y);
    
    return 0;
  }

In the above example:

  • The variables x and y are initialized with the values 5 and 10, respectively.
  • The assignment operator += is used to add the value of y to x and update the value of x accordingly.
  • The assignment operator *= is used to multiply the value of y by 2 and update the value of y accordingly.
  • The printf function is used to print the values of x and y.

Bitwise Operator

Bitwise AND Operator (&)

The bitwise AND operator performs a bitwise AND operation on the binary representation of two operands.

Example:

#include <stdio.h>
      
      int main() {
        unsigned int a = 5;  // binary: 00000101
        unsigned int b = 3;  // binary: 00000011
        
        unsigned int result = a & b;
        
        printf("Result: %u\n", result);  // binary: 00000001
        
        return 0;
      }

Bitwise OR Operator (|)

The bitwise OR operator performs a bitwise OR operation on the binary representation of two operands.

Example:

#include <stdio.h>
      
      int main() {
        unsigned int a = 5;  // binary: 00000101
        unsigned int b = 3;  // binary: 00000011
        
        unsigned int result = a | b;
        
        printf("Result: %u\n", result);  // binary: 00000111
        
        return 0;
      }

Bitwise XOR Operator (^)

The bitwise XOR operator performs a bitwise exclusive OR operation on the binary representation of two operands.

Example:

#include <stdio.h>
      
      int main() {
        unsigned int a = 5;  // binary: 00000101
        unsigned int b = 3;  // binary: 00000011
        
        unsigned int result = a ^ b;
        
        printf("Result: %u\n", result);  // binary: 00000110
        
        return 0;
      }
