Variable Declaration

Rules for defining variables

           1. A variable can have alphabets, digits, and underscore.
           2. A variable name can start with the alphabet, and underscore only. It can't start with a digit.
           3. No whitespace is allowed within the variable name.
           4. A variable name must not be any reserved word or keyword, e.g. int, float, etc.

Valid Variable

            int a;  
            int _ab;

Invalid variable

            int a b;  
            int long;

Variable Declaration Example

Here is an example of variable declaration:

var quantity;
        var price = 10.99;
        var total = quantity * price;

In the above example:

  • quantity is declared without an initial value, so it is undefined initially.
  • price is declared and initialized with the value 10.99.
  • total is declared and assigned the result of multiplying quantity and price.
